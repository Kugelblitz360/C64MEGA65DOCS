Testing the Core for Release

The C64 Core for the MEGA65 has been put through extremely thorough compatibility testing. This makes updates so challenging: Every new feature might break compatibility. The full test regimen, especially concerning demos that use every known trick in the book, can be found here:

https://github.com/MJoergen/C64MEGA65/tree/master/tests

If you find an incompatiblity not shown here, please raise an issue in the Github.

https://github.com/MJoergen/C64MEGA65/issues
